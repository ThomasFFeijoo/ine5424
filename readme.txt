Para solucionar o projeto foi desenvolvido o arquivo simple_protocol.h que representa o protocolo, além de diversos arquivos testes.

Os testes devem ser executados um de cada vez e como aplicações do EPOS (e.g. make APPLICATION=port_test_success run).

Os seguintes arquivos de testes foram utilizados para validar a solução:
- port_test_success.cc
- port_test_error.cc
- test_threads.cc
- retry_test.cc
- nic_test.cc
